Stéréo Zoom : sert à afficher une image stéréo jpeg constituée de deux vues jointes côte à côte. De plus sert à zoomer dedans, de manière cohérente dans chacune des images destinées à chacun de nos yeux, en maintenant le parallélisme des déplacements.

Les touches ont les fonction suivantes :

X
Pour changer le mode de la vision stéréo. Le mode par défaut dépend de votre image. Soit un mode en vision croisée (en louchant). Soit le mode en vision parallèle (regard dans le vague) peut aussi être utilisé avec un casque de réalité virtuelle.
Ctrl
Dans le mode de vue stéréo zoom, il peut être constaté un décalage entre les deux vues. La touche Ctrl permet de les ajuster, avec le glisser-cliquer de la souris, en limitant le déplacement à une seule des deux.
Z
Pour changer le mode de la vue stéréo.Le mode par défaut est le mode vue générale, adapté à la taille de l'écran (zoom Fit). Le mode Zoom affiche un pixel image par pixel d'écran (zoom 100%).
Étant donné que l'image est souvent plus grande que l'écran, il est également possible de se déplacer dans la vue stéréo par cliquer-glisser avec la souris.
Notez que le zoom du navigateur peut être combiné avec la molette de la souris ou les raccourcis Ctrl + et Ctrl −.
Enfin, en mode zoom 100%, l'image peut être très grande, et donc longue à charger en mémoire ... patience.
+/−, flèches
Pour zoomer et se déplacer avec le clavier.
R
Réinitialise au zoom 100% et au positionnemnt central de l'image.
F
Pour se placer en plein écran. La touche Échap, ou Esc, permet de quitter le plein écran (voire même le bouton de retour en arrière sur les mobiles).
W
Pour se placer en pleine fenêtre. La touche Échap, ou Esc, permet de quitter la pleine fenêtre.

La page de configuration permet de définir comment reconnaitre les images à afficher en stéréo.
