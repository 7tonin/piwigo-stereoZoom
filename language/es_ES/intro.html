stereoZoom: Se utiliza para mostrar una imagen estéreo formada por un par de vistas, unidas una al lado de la otra.

Especialmente para acercar y alejar, de forma coherente en cada una de las vistas previstas para cada uno de nuestros ojos, manteniendo el paralelismo de los desplazamientos.

El movimiento y el zoom se registran para cada imagen en una cookie, con el fin de volver al mismo lugar al volver a la imagen. Los datos se almacenan durante 7 días.
